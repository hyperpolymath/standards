# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Planned
- Additional data structures (HashMap, BinaryTree)
- Iterator implementations
- Async support

## [0.2.1] - 2025-11-22

### Fixed
- Improved documentation examples
- Fixed edge case in Queue::dequeue with empty queue

### Changed
- Updated dependencies to latest versions
- Improved error messages in tests

## [0.2.0] - 2025-11-15

### Added
- Optional serialization support via `serialization` feature
- Queue data structure with FIFO behavior
- Comprehensive test suite with >70% coverage
- API documentation with examples
- Basic usage example

### Changed
- Restructured module organization for better clarity
- Enhanced Stack implementation with additional methods

## [0.1.0] - 2025-11-01

### Added
- Initial release
- Stack data structure with basic LIFO operations
- Core functionality: push, pop, peek
- Basic test suite
- MIT/Apache-2.0 dual licensing
