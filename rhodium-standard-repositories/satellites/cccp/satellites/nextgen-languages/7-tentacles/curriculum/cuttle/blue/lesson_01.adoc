== What You Learned (Detective's Secret!)

You discovered:

* **Evidence:** Facts that point to answers
* **Deduction:** Using clues to figure things out
* **Logic:** If this is true, then that must be true

**In programming, we use logic to prove our code is correct - just like solving a mystery!**
