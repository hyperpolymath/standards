= Magic Indigo's Spell Cards
:toc:
:icons: font

*A transformation card game for 2-5 players, ages 8+*

== Overview

Cast spells to transform objects! Combine simple spells into powerful combos. But beware - spells have rules, and casting wrong fizzles your magic!

*Players:* 2-5
*Time:* 20-30 minutes
*Ages:* 8+ (Cuttle stage)

== What's In The Box

* 30 Object Cards (things to transform)
* 40 Spell Cards (transformations)
* 10 Combo Cards (multi-step spells)
* 6 Fizzle Cards (failed spells)
* 1 Cauldron Board (center play area)
* Rules card

== Setup

1. Shuffle Object cards, deal 3 to each player
2. Place 5 Objects face-up in the Cauldron (center)
3. Shuffle Spell cards, deal 5 to each player
4. Keep Fizzle deck separate

== How To Play

=== On Your Turn

1. **Cast** a spell on an Object (yours, cauldron, or opponent's)
2. **Transform** the object if spell matches
3. **Score** if you create a target combination
4. **Draw** back to 5 spell cards

Game ends when Spell deck is empty.

== The Secret (Don't Tell the Kids!)

[NOTE]
====
*What they're learning:*

* **Transformation rules:** Input type â†’ Output type
* **Valid operations:** Not every spell works on every object
* **Composition:** Chaining transformations
* **Compile-time checking:** Invalid spells "fizzle" before executing

This is metaprogramming!

----

macro melt(input: Stone) -> Water

// This works:

----

Indigo is teaching macros and compile-time computation!

== Variants

=== Spell Crafting
Players can create NEW spells by combining two basic spells. Write on blank cards!

=== Duel Mode
Players target each other's objects. Transform opponent's objects to score!

=== Cooperative Cauldron
Everyone works together to transform a starting object through ALL types in order.
