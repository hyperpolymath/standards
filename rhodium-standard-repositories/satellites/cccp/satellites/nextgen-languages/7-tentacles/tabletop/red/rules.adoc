== Overview

Race to build the fastest valid path from START to FINISH! But watch out - some paths are blocked, and invalid paths don't count!

*Players:* 2-4
*Time:* 15-20 minutes
*Ages:* 8+ (Cuttle stage)

== Setup

1. Each player takes a START card
2. Shuffle Path, Obstacle, and Boost cards together
3. Deal 5 cards to each player
4. Place remaining cards as draw pile
5. Youngest player goes first

== How To Play

=== On Your Turn

1. **Draw** 1 card from the pile
2. **Play** 1-3 cards to build your path
3. **Discard** down to 7 cards if needed

=== Stealing & Blocking

* Play an Obstacle (ğŸª¨) on another player's path to block them!
* They must discard cards to go around
* You can only block once per turn

== Winning

First player to build a **valid path** from START to FINISH wins!

*Tiebreaker:* Fewest cards used (faster path!)

== Scoring Variant (For Longer Games)

Play 3 rounds. Score each round:

* **Finish first:** 5 points
* **Valid path:** 3 points
* **Each Boost used:** +1 point
* **Each Obstacle avoided:** +1 point

Highest score after 3 rounds wins!

== Variants

=== Tiny Track (Younger Kids)
Use a 3x3 grid. Remove obstacles. First to finish wins.

=== Maze Mode (Older Kids)
Pre-place obstacles on the board. All players navigate the same maze.

=== Speed Mode
Use a timer! 30 seconds per turn. Invalid paths on time-out.
