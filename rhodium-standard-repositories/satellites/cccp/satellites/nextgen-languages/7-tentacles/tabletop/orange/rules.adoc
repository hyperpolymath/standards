= Juggler Orange's Timing Game
:toc:
:icons: font

*A rhythm card game for 2-4 players, ages 8+*

== Overview

Keep all the balls in the air! Play cards in the right ORDER and TIMING to juggle successfully. Drop a ball and you're out!

*Players:* 2-4
*Time:* 15-20 minutes
*Ages:* 8+ (Cuttle stage)

== What's In The Box

* 40 Ball Cards (4 colors √ó 10 each)
* 12 Action Cards (Throw, Catch, Pass)
* 8 Timing Cards (Now, Wait, Rush)
* 4 Juggler Mats (tracks balls in air)
* Rules card

== Setup

1. Each player takes a Juggler Mat
2. Shuffle Ball and Action cards together
3. Deal 5 cards to each player
4. Place 3 balls (any color) on each mat's "Ready" zone

== How To Play

=== The Rhythm

Play proceeds in BEATS. Everyone plays simultaneously!

----
Beat 1: Everyone plays one card
Beat 2: Resolve all cards (order matters!)
Beat 3: Check for drops
Repeat!
----

=== On Each Beat

1. **Choose** a card from your hand (face down)
2. **Reveal** all cards simultaneously
3. **Resolve** in order: Throws ‚Üí Catches ‚Üí Passes

=== Timing Cards

**‚ö° Rush:** Your action happens FIRST this beat
**‚è∏Ô∏è Wait:** Your action happens LAST this beat
**üéØ Now:** Normal timing (default)

=== Dropping Balls

A ball DROPS if:
* It's been in Air for 2 beats AND
* You don't play a Catch card

**When you drop:**
* Remove that ball from the game
* Draw 2 cards (penalty)

**If you drop all 3 balls = YOU'RE OUT!**

== Winning

Last player with balls in play wins!

*Variant scoring:*
* +1 point per ball remaining
* +2 points per opponent's dropped ball you caused (via Pass)

== The Secret (Don't Tell the Kids!)

[NOTE]
====
*What they're learning:*

* **Sequencing:** Actions must happen in order
* **Timing:** Some things must happen before others
* **Concurrent events:** Multiple things happening at once
* **Resource management:** Balls = limited resources

This is concurrency!

----
// In code terms:
async function juggle() {
  await throw(ball1);    // Must complete before...
  await catch(ball1);    // ...catching!
  // Race condition: what if timing is wrong?
}
----

Orange is teaching async/await and scheduling!

== Variants

=== Cooperative Mode
All players share one pool of balls. Everyone wins or loses together!

=== Speed Mode
Use a metronome or timer. 5 seconds per beat!

=== Chain Juggling
Balls must be passed in a circle. Add coordination challenge!
