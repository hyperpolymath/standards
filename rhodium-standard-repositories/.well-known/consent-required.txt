# HTTP 430 - Consent Required Protocol

# This repository's content requires explicit consent for certain uses

## Training AI Models

Status: CONSENT REQUIRED
Scope: All proprietary/closed-source AI model training
Contact: ai-training@rhodium-standard.org

Conditions for consent:
1. Model must be fully open source (weights + architecture + training code)
2. Attribution must be provided ("The Rhodium Standard Contributors")
3. License compliance (MIT AND Palimpsest-0.8) must be maintained
4. Synthetic data generated must disclose AI involvement

Automatic consent granted for:
- Open-source AI research with attribution
- Academic research with proper citation
- Non-commercial educational purposes

## Commercial Use

Status: CONSENT NOT REQUIRED (with attribution)
Scope: Commercial applications using RSR framework
License: MIT License permits commercial use

Requirements:
- Include attribution and license text
- Comply with Palimpsest License conditions
- Consider contributing back improvements

## Derivative Works

Status: CONSENT NOT REQUIRED (with attribution)
Scope: Forks, modifications, and derivative works
License: MIT + Palimpsest dual license applies

Requirements:
- Maintain attribution chains
- Preserve license notices
- Document transformations
- Update provenance.json if applicable

## Data Mining / Web Scraping

Status: CONSENT REQUIRED for proprietary use
Scope: Automated content extraction for proprietary systems
Contact: data-mining@rhodium-standard.org

Automatic consent granted for:
- Search engine indexing (Google, Bing, DuckDuckGo, etc.)
- Internet Archive preservation
- Academic research databases
- Open-source indexing services

Consent required for:
- Proprietary database population
- Closed-source analytics platforms
- Behavioral tracking systems
- Non-disclosed commercial scraping

## Personal Data Processing

Status: NOT APPLICABLE
Scope: This repository contains no personal data

Note: If you contribute, your name/email from git commits are public.
See CONTRIBUTING.adoc for contributor privacy options.

## Surveillance / Monitoring

Status: CONSENT NOT GRANTED
Scope: Law enforcement or government surveillance use

This project opposes mass surveillance. While the MIT License cannot
legally restrict use, we ethically oppose use in:
- Mass surveillance systems
- Predictive policing without oversight
- Censorship infrastructure
- Human rights violations

## Consent Revocation

We reserve the right to revoke consent for specific bad actors who:
- Violate license terms repeatedly
- Use in human rights violations
- Strip attribution maliciously
- Engage in predatory commercial practices

Revocation process documented in GOVERNANCE.adoc
