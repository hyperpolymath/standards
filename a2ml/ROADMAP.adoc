= A2ML Roadmap

== Current Status

Design phase complete. v0.6.0 draft includes hardened Module 0 parsing,
aligned vectors, refreshed docs, demo build, and core tests passing.

== Milestones

=== v0.1.0 - Spec Freeze (Draft)
* [x] Surface grammar draft
* [x] Typed core outline
* [x] Module 0 quickstart
* [x] Formal grammar appendix (EBNF + tokens)
* [x] Conformance and test vectors
* [x] Release notes + announcement draft

=== v0.2.0 - Reference Parser (Prototype)
* [x] Module 0 parser (surface -> AST) (prototype in `prototype/rescript/`)
* [x] HTML renderer (basic) (prototype in `prototype/rescript/`)
* [x] Lax/checked mode validation (basic)
* [x] Publish web demo page
* Target: 2026-03

=== v0.3.0 - Attestation Core
* [ ] Idris2 typed core encoding
* [ ] Proof obligations for invariants
* [ ] Opaque payload round-trip tests
* Target: 2026-05

=== v0.6.0 - Prototype Hardening
* [x] Directive parsing tightened (`@refs`, `@opaque`, attrs)
* [x] Vector suite aligned and passing
* [x] CLI docs and help updated
* [x] Web demo published
* [x] Core tests runnable via `just core-tests`
* Target: 2026-02

=== v0.4.0 - Web Rendering
* [ ] JS/Rescript parser/renderer
* [ ] Web component for client-side rendering
* [ ] WASM demo page (rescript-wasm-runtime)
* Target: 2026-07

=== v1.0.0 - Stable
* [ ] Spec stabilization and IANA submission
* [ ] Compatibility tests across renderers
* [ ] Formal conformance suite
* Target: 2026-10
