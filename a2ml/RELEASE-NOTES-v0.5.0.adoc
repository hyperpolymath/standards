= A2ML v0.5.0 Release Notes
:toc:
:toclevels: 2

== Summary

v0.5.0 advances the prototype parser, inline AST, directive mapping, and
translator coverage, and formalizes inline rules in the spec.

== Highlights

* Inline AST parser and renderer
* Directive attribute rendering in HTML
* Translator support for fig/table/refs directives
* New vectors for refs, tables, and inline edge cases
* Inline parsing rules documented
