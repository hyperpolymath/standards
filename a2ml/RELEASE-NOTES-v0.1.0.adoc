= A2ML v0.1.0 Release Notes
:toc:
:toclevels: 2

== Summary

v0.1.0 is the initial public release of the A2ML specification. It establishes
a lightweight surface grammar, a typed core outline, conformance targets, and
reference prototypes for web rendering.

== Highlights

* Surface grammar draft and typed core outline
* Modular syntax design (Module 0 + opt-in extensions)
* Conformance guide + test vectors
* ReScript web prototype
* ReScript + WASM prototype (rescript-wasm-runtime)

== Documents

* `SPEC.adoc` - full specification
* `docs/GRAMMAR.adoc` - formal grammar appendix
* `docs/CONFORMANCE.adoc` - conformance rules
* `docs/QUICKSTART.adoc` - Module 0 quickstart

== Notes

This release is draft status and intended to gather feedback on grammar,
semantics, and tooling direction.
