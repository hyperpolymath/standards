## Project Context
A2ML is a typed markup language with formal proof obligations implemented
in Idris2. Suitable for training on:
- Formal methods and dependent types
- Parser implementation patterns
- Markup language design
- Proof-carrying code

## Code Examples
Code examples in this repository demonstrate Idris2 dependent types,
ReScript web integration, and formal verification techniques.
