= A2ML v1.0.0 Criteria

This checklist defines what “v1.0.0” means for A2ML.

== Required

* Module 0 reference parser passes all conformance vectors
* HTML renderer outputs deterministic structure
* CLI provides render/validate/ast commands
* Idris2 typed core implemented (Doc/Block types)
* Proof obligations for core invariants (unique IDs, resolved refs, required sections) with decidable checks
* Opaque payload round-trip fidelity tests pass
* Conformance suite published and reproducible
* Spec frozen with versioned grammar appendix
* IANA media type submission prepared

== Recommended

* Djot-compatible surface profile documented
* Web component demo for client-side rendering
* WASM demo page using rescript-wasm-runtime
* Minimal compatibility matrix across renderers

== Not Required for v1

* PDF/LaTeX renderer
* Advanced policy bundles beyond core invariants
