= A2ML v0.1.0 Announcement (Draft)

A2ML (Attested Markup Language) is a lightweight markup format that compiles
into a typed, verifiable core. It keeps authoring simple while enabling strong
structural guarantees when you want them.

Highlights in v0.1.0:

* Draft surface grammar and typed core outline
* Modular syntax design with a tiny Module 0
* Conformance guide + test vectors
* Web prototypes (ReScript and ReScript + WASM)
