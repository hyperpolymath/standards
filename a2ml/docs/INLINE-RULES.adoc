= Inline Parsing Rules (Prototype)

This document describes the current inline parsing rules for the prototype.

== Supported

* Strong: `**text**`
* Emphasis: `*text*`
* Links: `[label](url)`

== Notes

* Parsing is non-nested and left-to-right.
* Unclosed markers are treated as literal text.
