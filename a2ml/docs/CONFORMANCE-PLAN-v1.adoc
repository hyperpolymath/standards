= A2ML v1 Conformance Plan (Draft)
:toc:
:toclevels: 2

This plan defines the evidence required to claim v1 conformance.

== Scope

* Module 0 surface parser
* Checked-mode validator
* HTML renderer
* Typed core translation (Idris2)

== Evidence Artifacts

* `build/vector-report.json` from `just vectors-report`
* Parser AST dumps for representative inputs (`just dump-ast <file>`)
* Determinism checks (same input -> same AST/HTML)
* Typed core translation tests (Idris2)

== Test Matrix

[cols="1,3,1",options="header"]
|===
| Area | Requirement | Status
| Vectors | All v0 vectors pass | pending
| Determinism | Stable HTML output | pending
| Checked mode | Reject unresolved refs | pending
| Opaque | Byte-for-byte preservation | pending
| Typed core | Idris2 translation compiles | pending
|===

== Execution

[source,bash]
----
just vectors-report
just dump-ast tests/vectors/module0-basic.a2ml
just core-tests
----

* Conformance status is reported in `docs/CONFORMANCE-REPORT-v1.adoc`.
